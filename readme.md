Compta
======

A simple C++ application to read and parse
a simple ascii input files, containing
comptability information.

Intallation
===========

From source files:
  - cd $SRC_DIR
  - ./bootstrap
  - cd ../
  - mkdir build
  - cd build
  - ../$SRC_DIR/configure
  - make
  - make check (optional)
  - make install (may require root privileges)
